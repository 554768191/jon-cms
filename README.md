# jon-cms
基于 springboot  Bootstrap3  thymeleaf 结合mybatis xml配置，搭建的一套CMS管理后台.jon-cms是一套通用易改造的系统。

1、**使用@UnLoginLimit注解来管控免登陆接口，在Controller方法上添加。** <br/>
2、**使用AdminInterceptors拦截器进行权限校验。** <br/>
3、**使用GlobalExceptionHandler统一异常处理。** <br/>
